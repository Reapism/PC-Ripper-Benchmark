Overview

     PC Benchmarking Tool is a computer diagnostic tool to which you can test the speed of your computer and it will be scored on a scale of 0 to 100. With this, it tests three main parts of your computer with various tests.

                When you first enter the application, on the right side will have all your computer specifications to the computer your running it on and all the details of it. It will show your Processor or CPU, RAM, DISK, and GPU specs. This is part of your welcome page when you sign in. On the left hand side, it will explain your profile; your name, your user type, and your experience level. This affects your the output of the program. From there, by pressing the button with four little squares, it will slide open and show you five buttons; the home page, CPU test page, RAM test page, DISK test page, and settings page. With that you can navigate the program. From the amount of time it took to do these for test, it runs it three times for each algorithm and averages it, before it gives you the time and the score of the benchmarking test.

CPU Test Page

  1. Successorship
  1. Counts from 1 to n number through a loop.

  1. Boolean Logic
  1. Compares two random numbers and returns which is greater for n number of times

  1. Queue
  1. Adds, removes, and searches random numbers from its contents for n number of times in the queue.

  1. Linked List
  1. Adds, removes, and searches from its contents for n number of times in a linked list

  1. Binary Tree
  1. Adds, removes, and searches random numbers from its contents for n times, in a Tree data structure.

RAM Test Page

  1. Virtual Folder Matrix
  1. Creates n number of virtual directories in memory and with about half of n of the files will have files in them and this all will be read back to memory

  1. Virtual Bulk File
  1. Creates n number of virtual files in memory before reading them back

  1. Reference Dereference Parse
  1. Creates ‘RipperFiles’ in memory, finds them, and deletes them

DISK Test Page

        

  1. Folder Matrix
  1. Generates folders in your chosen directory and reads them back

  1. Bulk File
  1. Creates files and writes data into them in the directories that Folder Matrix creates

  1. Read and Write Parse
  1. Writes a single large file in your chosen directory and reads them back into memory from the file system

  1. Disk Ripper
  1. Runs the Read/Write Parse test in every directory that Folder Matrix creates
